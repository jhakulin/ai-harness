# Claude Instructions

This repository stores reusable software-development agent skills and future agent descriptions.

Follow the same source files used by Codex:

- Skill guideline: `guidelines/skill-guideline.md`
- Agent guideline: `guidelines/agent-guideline.md`
- Skills: `skills/<name>/SKILL.md`
- Shared agent instructions: `AGENTS.md`

Keep additions small. Do not introduce Claude-specific commands, plugins, hooks, or duplicated skill copies unless explicitly requested.
